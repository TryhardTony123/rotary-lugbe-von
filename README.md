# Rotary Club of Lugbe VON

A responsive Rotary Club of Lugbe VON management application built with C# and Blazor, focused exclusively on member attendance and birthdays.

## Run locally

```powershell
dotnet run
```

Open the local address shown in the terminal.

## Deploy on Render

The repository includes a production `Dockerfile` and `render.yaml`. Connect the
repository to Render, create a Blueprint, and select the Starter web-service plan.
